1. Подготовить список поликлиник доступных для работы через sevmed интерфейс и их URL:PORT данные
2. Научиться парсить данные со страниц поликлиник, с тем чтобы формировать интерфейс для отслеживания специальностей/врачей

- научиться в интерфейсе ходить вперёд и назад по кнопкам.

3. Подготовить два уровня работы с запросами, в зависимости от конфигурации (в дальнейшем это будут два разных приложения):
  - первый уровень: отслеживание врачей по специальности. Т.е. нам нужен любой врач по специальности.
  - второй уровень: выбираем специальность и затем есть возможность выбрать конкретного врача (отслеживать и получать уведомления)
